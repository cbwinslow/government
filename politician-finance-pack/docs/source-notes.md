# Source-backed notes

## Campaign finance
- OpenFEC API documentation: https://api.open.fec.gov/developers/
- FEC browse/bulk data portal: https://www.fec.gov/data/browse-data/
- OpenFEC is a RESTful web service for FEC data.

## House financial disclosures
- House Ethics financial disclosure page says FD reports and PTRs must be filed with the Clerk of the House and may be filed using the electronic system at https://fd.house.gov.
- House Clerk public disclosure site provides access to financial disclosure materials and related public disclosure documents.

## Periodic transaction reports
- DOI/OGE FAQ states PTRs are required when filers make certain financial transactions such as buying or selling stock, and absent extension they are due no later than 45 days after the transaction occurs.

## Legislative APIs
- Congress.gov API v3: API key required, XML/JSON, rate limit 5,000 requests/hour, default 20 and max 250 records per request.
- GovInfo: API, bulk data, feeds, sitemaps, and developer hub available.
